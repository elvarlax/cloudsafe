# Cloudsafe

## Installation

1. Install Anaconda
2. Open Anaconda Prompt
3. conda config --prepend channels conda-forge
4. conda create -n ox --strict-channel-priority osmnx
5. conda activate ox
6. Open Anaconda Navigator
6. Install Jupyter Notebook
7. pip install -r requirements.txt
